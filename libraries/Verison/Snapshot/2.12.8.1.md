一个小更新，修复一些 2.12.8.0 的问题，看看这次能不能上正式版……

## 优化
- 当被 Xbox 封禁账号时，会给出明确的提示（#8655，@LinQingYuu）
- 在搜索社区内容前，会将繁体中文转换为简体再进行匹配（#8657，@GotohHitori52）
- 添加“刷新披风列表”选项
- 若删除到回收站失败，会回退到永久删除
- 除特定几个“可点击”表述不明显的控件外，其余控件改回普通鼠标样式，而非手形

## Bug 修复
- 若缺少特定系统文件夹，会导致 PCL 无法删除文件夹（#8636，@qwer154899）
- 可能无法更换披风（#8638，@kiun114514，PR @LinQingYuu）
- 如果手速快，可以同时打开多个 PCL（#8651，PR @LinQingYuu）
- 无法同时启动 3 个特定版本的游戏
- 由于出现了一个在 Java 21 不使用 JLW 导致崩溃的案例，现在在新版本 Java 也会尝试使用 JLW
- 更改皮肤失败的提示信息有误
- 土豆码校验可能失败

## API 更改
- 导出整合包添加“协议”选项，以导出协议文件（#8300，@19136644525lxy）
- 导出整合包的规则现在忽略大小写
- 新替换标记： `{pcl_sha1}` ，PCL exe 文件的 SHA1

## 技术性更改
- 迁移加密相关代码到 [MeloongCore.CryptographyUtils](https://github.com/Meloong-Git/MeloongCore/blob/master/Shared/Utils/CryptographyUtils.cs)
